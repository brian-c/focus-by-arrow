FocusByArrow

Move focus from one element to another based on where they sit on the screen.

This is generally useful for lists of links and buttons in menus. The default arrow key action is prevented for non-text inputs.

Usage: 

        
          // Start listening for arrow key presses within an element:
          var fba = new FocusByArrow('.some-container');

          // Stop listening:
          fba.stopListening();

          // Re-start listening:
          fba.startListening();
